## Release History

<Update label="2024-10-15" description="v2.1.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time collaboration features for team editing
- Introduced advanced search with filters and facets
- Implemented custom domain support for enterprise customers
- Added support for Mermaid diagrams in documentation

## Improvements

- Enhanced API performance with response caching
- Improved mobile responsiveness across all pages
- Updated authentication flows with better error handling
- Optimized image loading with lazy loading

</Update>

<Update label="2024-09-20" description="v2.0.0" tags={["breaking", "feature"]}>

## New Features

- Launched new API v2 with improved endpoints
- Added role-based access control for teams
- Introduced documentation templates library
- Implemented webhook integrations for external services

## Breaking Changes

- API v1 endpoints are now deprecated; migrate to v2
- Changed authentication token format for enhanced security
- Removed support for legacy browsers (IE 11 and below)

</Update>

<Update label="2024-08-10" description="v1.5.0" tags={["feature", "bugfix"]}>

## New Features

- Added export functionality for PDF and HTML formats
- Introduced comment system for collaborative feedback
- Implemented dark mode theme option
- Added support for multiple languages

## Bug Fixes

- Fixed issue with image uploads failing on large files
- Resolved search indexing problems for new documents
- Corrected permission inheritance in team hierarchies
- Fixed mobile layout issues on iOS devices

</Update>
